# 💻 Coderly >_<
A quick and easy way to run Coderly directly from your terminal.

## 🚀 How to Run

Run on PowerShell Copy and paste the command below into your PowerShell window and press Enter:
```python
Invoke-RestMethod -Uri "https://raw.githubusercontent.com/Bruhrbx/Coderly/refs/heads/main/run.ps1" | Invoke-Expression
```
